📋 How to Connect to Google Sheets (click to expand)
Step 1 — Create a Google Sheet
Open Google Sheets, create a new spreadsheet, and name it "Reliable VA — Subscribers". Add these column headers in Row 1:
Timestamp | First Name | Last Name | Email | Phone | Business | Newsletter | Sales | Reminders

Step 2 — Create an Apps Script
In your Sheet, go to Extensions → Apps Script. Replace all default code with: 
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data  = JSON.parse(e.postData.contents);
  sheet.appendRow([
    new Date().toLocaleString(),
    data.firstName,
    data.lastName,
    data.email,
    data.phone    || '',
    data.business || '',
    data.newsletter ? 'Yes' : 'No',
    data.sales      ? 'Yes' : 'No',
    data.reminders  ? 'Yes' : 'No'
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({result:'success'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet() {
  return ContentService.createTextOutput('OK');
}
Step 3 — Deploy as Web App
Click Deploy → New Deployment → Web app.
Set "Execute as" → Me and "Who has access" → Anyone. Click Deploy, then copy the Web App URL.

Step 4 — Paste the URL in this page
Find the line near the bottom of this HTML file that says:
const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE';
Replace the placeholder with your deployed URL. Save and publish the page. Done! 🎉
